# 🔐 Password Strength Analyzer

A modern **full-stack web application** that helps users evaluate the strength of their passwords and improve their online security through real-time analysis, security recommendations, and secure password generation.

---

## ✨ Features

- 🔍 Real-time password strength analysis
- 🎨 Color-coded strength indicators
- 💡 Security improvement suggestions
- 🔑 Strong password generator
- 📱 Responsive brown-themed user interface
- ⚡ Flask REST API backend
- 🌐 Interactive HTML, CSS & JavaScript frontend

---

## 🛠️ Tech Stack

- 🐍 Python
- ⚡ Flask
- 🌐 HTML5
- 🎨 CSS3
- 💻 JavaScript
- 🗂️ Git & GitHub

---

## 📊 Password Strength Levels

| Strength | Indicator |
|----------|-----------|
| 🔴 Weak | Easily guessable password |
| 🟡 Medium | Moderately secure password |
| 🟢 Strong | Secure password |
| 🟢 Very Strong | Highly secure password |
